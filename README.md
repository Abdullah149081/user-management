# User-Management

## Installation

To install all dependencies, use the following command:

```
yarn
```

## Run and build

To run the project in development mode:

```
npm run dev
```

To build the project:

```
npm run build
```

### Code formatting and Error handling
To fix errors:
```
npm run lint:fix 
```
To format code:

```
npm run prettier:fix
```
